Golang流程控制

程序的流程控制结构一共有三种：顺序结构、选择结构、循环结构

顺序结构：从上到下，逐行执行，所有程序的默认逻辑

选择结构：条件满足某些代码才会执行

  • if
  • switch
  • select

循环结构：条件满足某些代码会被反复执行0-N次

  • for

循环控制语句

循环控制语句可以控制循环体内语句的执行过程。

GO 语言支持以下几种循环控制语句：

控制语句 描述
break 语句 经常用于中断当前 for 循环或跳出 switch 语句
continue 语句 跳过当前循环的剩余语句，然后继续进行下一轮循环。
goto 语句 将控制转移到被标记的语句。

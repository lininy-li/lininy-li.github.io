Java复习

首先是jdk环境搭建以及开发软件，不多说，不会百度

然后创建个java文件

启动类就不写了，执行后得到一个Hello Wrod！的值。

值得新手注意的点：

代码的格式、英文的大小写、文件名、包名是否正确、目录啥的以及中英文形式的标点符号。

编译于运行

编译就是利用编译器（javac）将源程序编译成字节码à字节码文件名：源文件名.class

执行即利用虚拟机（解释器，java）解释执行class字节码文件。

了解了以上这些就去准备编码环境吧，开发软件不建议加汉化补丁哦

接下来就是基础语法了

从注释开始

平时我们编写代码，在代码量比较少的时候，我们还可以看懂自己写的，但是当项目结构一旦复杂起

来，我们就需要用到一个注释了，注释就类似于我们上学时候写的笔记，我们看着笔记就知道自己写的

什么东西了！在程序中也是如此。我们来看一下Java中的注释怎么写，看以下代码：

你只要知道的是注释并不会被执行，是写给编码的人看的，所以写注释是一个非常好的习惯，会大大的增加你代码的可读性

Java中注释一般分为三种：

单行注释：只能注释当前行，以//开始，只要不换行就不算结束

多行注释：以/开始，以/结束，因为md文档的特殊性导致第一个/后和第二个/前的*隐藏了，从代码上来看会更方便一些

文档注释：用于生产API文档，配合JavaDoc。

先说说命名吧

关于 Java 标识符，有以下几点需要注意：

  1. 所有的标识符都应该以字母（A-Z 或者 a-z）,美元符（$）、或者下划线（_）开始

  2. 首字符之后可以是字母（A-Z 或者 a-z）,美元符（$）、下划线（_）或数字的任何字符组合

  3. 不能使用关键字作为变量名或方法名。

  4. 标识符是大小写敏感的

  5. 合法标识符举例：age、$salary、_value、__1_value

  6. 非法标识符举例：123abc、-salary、#abc

  7. 可以使用中文命名，但是一般不建议这样去使用，也不建议使用拼音，一言难尽
